General

Why should I choose Mongoclient over other tools ?

Mongoclient is the simplest way to manage your MongoDB database and emphasis on end user's requests, and gives them to use full capabilities of modern browsers as well as desktop application. It's secure to use Mongoclient on production environments, since it's open source.

Do I pay anything to use Mongoclient ?

No, not at all. Mongoclient's commercial version is free to use and you can use it on your own company without paying anything. But if you need more than that and need support, in-house deployment or special feature implementation just contact us by sending a mail to info@mongoclient.com

What's the license of Mongoclient ?

Mongoclient uses AGPL 3.0 starting from version 2.0.0. But before version 2.0.0 the license was MIT

Can I buy support ?

Absolutely, you can. Contact us by sending a mail to info@mongoclient.com

Features

How to select only specific fields for find/findOne queries ?

Mongoclient offers query options for such features. When you select find query options will be rendered correspondingly. With project option you can exclude or include fields for your query. Check here for more info.

How Mongoclient interprets BSON types such as ObjectID and ISODates ?

As a huge ease of use, Mongoclient uses extended JSON to handle BSON data types. Check here for more info.

Can I manage admin database instead of connected database for admin queries ?

That's absolutely possible, you can click and check right upper corner checkbox to query admin database instead of the currently connected one. Check here for more info.

Can I manage admin database users instead of connected database's ?

That's absolutely possible just like admin queries, you can click and check right upper corner checkbox to use admin database instead of the currently connected one. Check here for more info.

Can I see my old queries ?

Mongoclient stores latest 20 queries for each collection. Check here for more info.

What Mongoclient uses to connect my database ?

At behind the scenes, Mongoclient uses NodeJS MongoDB driver and it's advantages for the all purposes. As an advantage of driver, Mongoclient uses connection pooling strategy. Check here for more info.

How do I use autocompletion ?

First of all, make sure you've enabled the autocompletion feature from settings page. Afterwards, you can use autocompletion on selectors by CTRL+SPACE buttons combination. Check here for auto completion setting.
